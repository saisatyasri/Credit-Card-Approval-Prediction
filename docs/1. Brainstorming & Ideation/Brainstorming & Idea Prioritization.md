## Justification for Selected Idea

The **Credit Card Approval Prediction** project was selected because:

1. **High Feasibility** — A publicly available Kaggle dataset (438K+ records) with real applicant financial data is readily accessible for training.
2. **High Importance** — Banks process thousands of credit applications daily; automation directly addresses a real business pain point.
3. **Full Stack Scope** — The project covers the complete ML workflow: data collection → preprocessing → model training → web deployment, satisfying all academic requirements.
4. **Team Skill Match** — The team has Python, ML, and web development skills that align well with the chosen tech stack (Flask, scikit-learn, XGBoost).
