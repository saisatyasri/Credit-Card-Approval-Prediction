About this Project

This project automates credit card approval decisions using machine learning. It trains four classifiers on historical applicant data from Kaggle and integrates the best model into this Flask web application. The pipeline covers data preprocessing, feature engineering, model evaluation, and real-time prediction via an intuitive UI.
